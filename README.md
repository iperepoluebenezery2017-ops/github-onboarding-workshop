## About This Repository

Welcome to the official GitHub onboarding and class-notes repository for the **Microsoft Naija Security User Group SC-500 Mentorship Program 2026**.

Every Saturday, a new security topic is covered by a mentor. This repo gives mentees a structured place to:

- Fork the repo and add their own notes after each session
- Track their learning progress across all 26 Saturday sessions
- Build a public portfolio of security knowledge they can share with employers

---

## How to Participate (Fork & Edit)

> **You do not need to be added as a collaborator. Each mentee gets their own file — no conflicts, no overwriting anyone else's work.**

### Steps

1. **Fork** — Click the **Fork** button at the top-right of this page to copy the repo to your own GitHub account
2. **Clone** your fork locally (optional) — `git clone https://github.com/YOUR-USERNAME/github-onboarding-workshop.git`
3. **Go to the session folder** — Navigate into `classes/` and open the folder for the session you attended (e.g. `session-01-2026-07-04-microsoft-entra/`)
4. **Copy the template** — Duplicate `notes-YOURNAME.md` and rename it to `notes-firstname-lastname.md` (e.g. `notes-oyimafu-emmanuel.md`)
5. **Fill in your notes** — Add your key takeaways, lab results, questions, and resources
6. **Commit** with a clear message, e.g. `docs: add session-01 notes - Oyimafu Emmanuel`
7. **Push** to your fork, then **open a Pull Request** — the facilitator reviews your file before merging into the main repo

## Class Notes Folder

All session files live in the [`classes/`](./classes/) folder. Each file is pre-filled with the session details and has blank sections for you to fill in after attending:

- **Key Concepts** — What was covered
- **Lab Notes** — Your hands-on work and results
- **My Takeaways** — What you personally learned
- **Questions** — Anything you want to follow up on
